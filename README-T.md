# 霓虹蛇 | Neon Snake Game

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)

這是一個充滿未來感、霓虹主題的經典貪吃蛇遊戲，使用原生 JavaScript 和 HTML5 Canvas 構建。具有動態視覺效果、粒子系統和復古合成音效引擎。

## 🌟 功能特色

- **霓虹美學**：發光的視覺效果、掃描線效果和賽博龐克風格的配色。
- **動態視覺**：
  - 隨時間變化的彩虹漸變蛇身。
  - 吃到食物時的粒子爆炸效果。
  - 遊戲結束時的畫面震動效果。
  - 流暢的動畫和過場。
- **復古音效引擎**：
  - 自定義 Web Audio API 實作（無外部資源）。
  - 程序化生成的晶片音樂背景音樂。
  - 遊戲動作的合成音效。
- **遊戲功能**：
  - 經典貪吃蛇遊戲機制。
  - 本地排行榜系統記錄高分。
  - 登入/訪客模式（模擬驗證）。
  - 支援桌面和行動裝置的響應式設計。

## 🎮 操作方式

遊戲支援鍵盤和觸控/點擊操作。

| 動作 | 鍵盤按鍵 |
|--------|--------------|
| **向上移動** | `上箭頭` / `W` |
| **向下移動** | `下箭頭` / `S` |
| **向左移動** | `左箭頭` / `A` |
| **向右移動** | `右箭頭` / `D` |
| **暫停/繼續** | `空白鍵` |

## 🚀 如何執行

由於這是一個靜態網頁應用程式，您可以直接在瀏覽器中執行它。

### 方法 1：直接開啟
只需雙擊 `index.html` 檔案即可在預設瀏覽器中開啟。

### 方法 2：本地伺服器（推薦）
為了獲得最佳體驗（特別是為了符合音訊自動播放策略），建議使用本地開發伺服器來執行檔案。

如果您已安裝 Python：
```bash
# Python 3
python -m http.server 8000
```
然後在瀏覽器中前往 `http://localhost:8000`。

或者，如果您使用 VS Code，可以使用 "Live Server" 擴充功能。

## 🛠️ 使用技術

- **HTML5 Canvas**：用於高效能 2D 渲染。
- **CSS3**：用於 UI 樣式、動畫和霓虹發光效果。
- **Vanilla JavaScript (ES6+)**：核心遊戲邏輯和狀態管理。
- **Web Audio API**：用於音樂和音效的即時音訊合成。

## 📂 專案結構

- `index.html`：主要入口點和 UI 結構。
- `style.css`：所有視覺樣式，包括動畫和響應式佈局。
- `snake-game.js`：核心遊戲引擎、渲染循環和碰撞檢測。
- `audio-manager.js`：用於生成聲音和音樂的音訊合成器。
- `data-manager.js`：處理資料持久化（分數、使用者設定檔）。
- `app.js`：連接 UI 與遊戲邏輯並處理事件監聽器。
